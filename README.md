##  Pizzaria API (FastAPI)

API REST de uma pizzaria desenvolvida em Python utilizando FastAPI, criada como projeto prático de estudo para aprofundar conhecimentos em FastAPI.
O projeto aborda conceitos fundamentais de construção de APIs, autenticação, organização de rotas e integração com banco de dados utilizando SQLAlchemy.

### Tecnologias utilizadas
- Python
- FastAPI
- SQLAlchemy
- JWT Authentication
- Pydantic
- Uvicorn
- Swagger/OpenAPI

### Objetivos do projeto
- Aprender os fundamentos do FastAPI
- Criar APIs
- Trabalhar com banco de dados usando ORM
- Estruturar um backend de forma organizada
- Documentar endpoints automaticamente com Swagger
